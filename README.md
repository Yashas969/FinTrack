# FinTrack - Personal Finance Tracker

A modern, responsive personal finance tracking prototype built with React, Tailwind CSS, and Recharts.

## Features

- **Mock Authentication**: Login with any username/password.
- **Dashboard**: Visual overview of your finances with charts.
- **Income & Expense Tracking**: Add, view, and delete transactions.
- **Data Persistence**: All data is saved to `localStorage`.
- **Responsive Design**: Works on desktop (Sidebar) and mobile (Bottom Navigation).

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Visit `http://localhost:5173`

## Tech Stack

- React + Vite
- Tailwind CSS
- Lucide React (Icons)
- Recharts (Charts)
- React Router DOM
